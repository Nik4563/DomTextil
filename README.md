## 📖 О проекте

**ДомТекстиля** — учебный проект многостраничного интернет-магазина текстиля для дома. Сайт создан с нуля с использованием чистого HTML и CSS и охватывает все основные разделы, характерные для реального e-commerce магазина.

Магазин «ДомТекстиля» был основан в 2000 году как семейный бизнес и вырос в узнаваемый бренд с командой более 280 профессионалов. Миссия магазина — сделать комфортный сон и уютный дом доступными для каждого.

## 🛒 Как сделать заказ

1. Выбрать товар из каталога
2. Добавить в корзину
3. Оформить заказ (имя, телефон, адрес)
4. Оплатить удобным способом
5. Дождаться доставки
